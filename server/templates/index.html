What is MCP?

Model Context Protocol (MCP) is a protocol for extending AI assistants with external tools and data sources.

With MCP, you can create custom tools that AI assistants can use to perform actions like:

  • Reading and writing files
  • Accessing databases
  • Making API calls
  • Running custom code
  • And much more!
